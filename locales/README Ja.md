🌍 インターフェースのローカライゼーション
このフォルダには、RimWorld Translator Grabber 用の翻訳ファイルが含まれています。

📁 ファイル構造
各言語はシンプルな名前の個別ファイルに保存されます：
locales/
├── ru.json      # Русский
├── en.json      # English
├── ua.json      # Українська
├── ja.json      # 日本語
└── README.md    # このファイル

🚀 新しい言語を追加する方法（5 分）
**ステップ 1: 翻訳ファイルを作成**
既存のファイル（例：`en.json`）をコピーし、`{言語コード}.json` にリネームします：
# ドイツ語の例：
copy locales\en.json locales\de.json

主な言語コード：
- `de` — Deutsch（ドイツ語）
- `fr` — Français（フランス語）
- `es` — Español（スペイン語）
- `it` — Italiano（イタリア語）
- `pl` — Polski（ポーランド語）
- `zh` — 中文（中国語）
- `ko` — 한국어（韓国語）

**ステップ 2: 値を翻訳**
ファイルを開き、値（右側）のみを翻訳してください。キーは変更しないでください！
{
  "de": {
    "app_title": "RimWorld Translator Grabber",  ← 翻訳済み
    "menu_file": "Datei",                        ← 翻訳済み
    "menu_exit": "Beenden",                      ← 翻訳済み
    ...
  }
}
❌ 禁止：
{
  "menu_file": "Datei",  ← キーが変更されています — ダメ！
}
✅ 許可：
{
  "menu_file": "Datei"  ← キーは維持、値のみ翻訳 — OK！
}

**ステップ 3: ファイルを検証**
python -c "import json; json.load(open('locales/de.json',encoding='utf-8')); print('✅ ファイルは有効です！')"
または拡張検証：
python scripts/validate_locales.py locales/de.json

**ステップ 4: テスト**
- アプリケーションを起動
- 🌐 インターフェース言語メニューを開く
- 新しい言語を選択
- すべてのテキストが翻訳されているか確認

**ステップ 5: PR を送信**
翻訳ファイルを含めて Pull Request を作成してください。

🔧 自動インストール
`locales/{コード}.json` ファイルが存在すれば、新しい言語は自動的に表示されます。
追加の登録は不要です！

📝 翻訳ルール
**フォーマット**
✅ UTF-8 エンコーディングを使用
✅ 絵文字を保持（📦, ✅, 🌐 など）
✅ 行末のスペースを保持（元ファイルにある場合）

**特殊文字**
- `\n` — 改行
- `\"` — 文字列内の引用符
- `\\` — バックスラッシュ

**翻訳ステータス**
ファイルの先頭にコメントを追加できます：
{
  "_meta": {
    "language": "de",
    "translator": "YourName",
    "last_updated": "2026-04-05",
    "completion": "100%"
  },
  "de": {
    ...
  }
}

🐛 トラブルシューティング
**翻訳が読み込まれない**
- 原因: JSON ファイルが無効
- 解決: `python -m json.tool locales/de.json` でファイルを検証

**一部の文字列が翻訳されていない**
- 原因: キーが不足
- 解決: `en.json` と比較し、不足しているキーを追加

**テキストが切り捨てられる**
- 原因: 翻訳が長すぎる
- 解決: テキストを短くするか、略語を使用

💡 ヒント
- 元のファイルをテンプレートとして使用 — ゼロから始めない
- コンテキストが重要 — キーで文字列の使用箇所がわかります：
  - `menu.*` — メニュー項目
  - `tab.*` — タブ名
  - `editor.*` — 翻訳エディタ
  - `dialog.*` — ダイアログウィンドウ
  - `status.*` — ステータスバー
- 頻繁にテスト — 50〜100 行ごとにアプリを実行
- 絵文字を保持 — インターフェースの一部です
